Artificial Satellite Gateway

How It Works

A Python network application running on Raspberry Pi that functions as a Gateway.

The Gateway would receive zipcode and satellite identifier as input, make RESTful queries to Space-track API for required satellite data, make RESTFUL queries to NOAAAPI for weather conditions in zipcode area, calculate satellite visibility date/times in zipcode area, then give users notification about when the satellite is visible.

Space-Track API Query

Gateway makes RESTful queries to Space-Track API for required satellite data.

NOAA API Query

Gateway makes RESTful queries to NOAA API for weather conditions in zipcode area on satellite sighting date/time.
